Function Pointers in C:
  A pointer that stores the ADDRESS of a function.
  Declaration: return_type (*ptr_name)(parameter_types);
  Example:
    int (*fptr)(int, int);
    int add(int a, int b) { return a+b; }
    fptr = add;       // assign
    int res = fptr(3, 4);  // call via pointer -> res = 7
  USES:
    Callbacks: pass function as argument to another function.
    qsort(arr, n, sizeof(int), compare_func);
    Dispatch tables: array of function pointers for fast switching.
    State machines: function pointer per state.
    Plugin systems: dynamically load and call functions.
  typedef makes it cleaner:
    typedef int (*MathFunc)(int, int);
    MathFunc ops[] = {add, subtract, multiply};
    ops[0](3,4); // calls add(3,4)

What is an Array in C:
  An array is a COLLECTION of elements of the SAME data type stored in
  CONTIGUOUS (adjacent) memory locations.
  All elements share a common name and are accessed using an index.
  Why use arrays: instead of declaring 100 separate variables (a1, a2, ...a100),
    use one array: int a[100]; — much simpler and manageable.
  DECLARATION: data_type array_name[size];
    int marks[5];       -- 5 integers
    float temp[10];     -- 10 floats
    char name[20];      -- 20 characters (string)
  ACCESSING: array_name[index]  (index starts at 0, ends at size-1)
    marks[0] = first element, marks[4] = fifth element of marks[5].
  INDEX OUT OF BOUNDS: accessing marks[5] in an array of size 5 is UNDEFINED BEHAVIOUR.
  Arrays in C are zero-indexed: first element is at index 0, last at index (size-1).
